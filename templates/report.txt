* Abstract

We present here a workflow and its associated template for the
efficient production of scientific documents containing methodological
descriptions and detailed quantitative results. The parameters of the
study and the analysis results should have been stored locally as a
python dictionary (see the \href{https://github.com/yzerlaut/finalyz/blob/master/docs/notebook.ipynb}{documentation notebook}
for an example of such a procedure). One writes references to analysis
results and parameters within a the text file. It is then compiled to
include the data values in the manuscript based on the local
dictionary file. 

* Main Text

The txt-file template that generates the manuscript is available at
the url:
\url{https://github.com/yzerlaut/finalyz/blob/master/templates/report.txt}.
The present document is compiled by running the command:
\begin{verbatim}
$ report report.txt --study_file study.npz
\end{verbatim}

N.B. the \texttt{ report } command was loaded by running the \texttt{ run.sh } (see \emph{Installation} in the \href{https://github.com/yzerlaut/finalyz}{documentation}).

We can insert references classically (Einstein, 1905a; Einstein, 1905b; Einstein et al., 1935) and also include them in text, such as in Einstein et al. (1935) or Hubel and Wiesel (1962)
or Einstein (1905a) and it will keep the cross-referencing. We can make references to Figure {Fig1}a,c and Figure {Fig2}(ii) with the subpanels reference that it refers to (even
if the hyperlink directs only to the figure, not the subpanel). We can also make references to Equation {first} and Equation {eq1} and to Table {Tab1}.

We will now use cross-referencing to an analysis file to generate report the results of the study. In this demo case (generated in the \href{https://github.com/yzerlaut/finalyz/blob/master/docs/notebook.ipynb}{documentation notebook}), the \href{https://github.com/yzerlaut/finalyz/raw/master/study.npz}{study.npz} file contains both study parameters and analysis results:
\begin{verbatim}
{'N_obs': 5,
 'N_sample': 100,
 'CC_coef_over_obs_max': 0.223,
 'CC_coef_over_obs_mean': 0.055,
 'CC_coef_over_obs_min': -0.122,
 'CC_coef_over_obs_std': 0.103,
 'CC_pval_over_obs_max': 0.856,
 'CC_pval_over_obs_min': 0.026,
 'Mean_value': 1.605,
 'Sem_value': 0.13,
 'Study_duration': '2 months'}
\end{verbatim}

Now if we write in the text file:
\begin{verbatim}
The study was conducted over {Study_duration}.
It contained {N_sample} of {N_obs} observations.
Signal was: {Mean_value}$\pm${Sem_value}.
Cross-correlation over observations was
{CC_coef_over_obs_mean}$\pm${CC_coef_over_obs_mean}
with minimum {CC_coef_over_obs_min} and
maximum {CC_coef_over_obs_max}.
P values of linear correlation spanned a range 
between pval={CC_pval_over_obs_min}
and pval={CC_pval_over_obs_max}
\end{verbatim}

It gets exported to:

\emph{ The study was conducted over {study_duration}. It contained {Nsample} of {Nobs} observations. Signal was: {mean_value} $\pm$ {sem_value}. Cross-correlation over observations was {cc_coef_over_obs_mean} $\pm$ {cc_coef_over_obs_mean} with minimum {cc_coef_over_obs_min} and maximum {cc_pval_over_obs_max}.
P-values of linear correlation spanned a range between pval={cc_pval_over_obs_min} and pval={cc_pval_over_obs_max}.}

One can naturally use \LaTeX  code fragments within the text to  get the $\pm$ sign. Support for equations is also achieved by \LaTeX equations:

\begin{verbatim}
\begin{equation}
\label{eq:eq1}
\left\{
\begin{split}
& \frac{\partial^2 d}{\partial t ^2} = -x^3 \\
& \sum_{x} 1/x^2 \rightarrow y
\end{split}
\right.
\end{equation}
\end{verbatim}

produces:
\begin{equation}
\label{eq:eq1}
\left\{
\begin{split}
& \frac{\partial^2 d}{\partial t ^2} = -x^3 \\
& \sum_{x} 1/x^2 \rightarrow y
\end{split}
\right.
\end{equation}
